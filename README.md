# Azure App Service — Build & Deployments UX POC

A proof-of-concept for a modernized Azure App Service deployment management experience in the Azure Portal.

## What Is This?

The current Azure Portal deployment UX for App Service is functional but dated. This POC explores what a modern, developer-friendly deployment experience could look like — one that reflects how people actually work with CI/CD in 2026.

## Quick Start

```bash
npm install
npm run dev
```
